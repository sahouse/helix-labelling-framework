about=HELIX Labelling Framework turns heterogeneous vector/raster EO labels into modular grid-aligned, time-aware, context-aware and uncertainty-aware supervision products for ML workflows. It provides QGIS Processing tools for preflight class schemas, spatial reconstruction to EO grids, temporal reconciliation, helical features, context/risk features, soft targets and export reports. Documentation is included in the docs folder and in the public repository README. Dependencies are QGIS/PyQGIS with GDAL and NumPy as provided by standard QGIS installations. The plugin is written in Python using the QGIS PyQt compatibility layer and is intended for Windows, Linux and macOS. Copyright (C) 2026 Sarah Hauser, Karlsruhe Institute of Technology (KIT).
